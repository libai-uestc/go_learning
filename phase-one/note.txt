go语言在以下情况自动加了分号
1. 如果一行它最后是一个标识符的话，比如像string、int这些标识符，它就会自动帮我们加分号
2. 如果一行的末尾是一个字面量的话，比如18，比如"libai"这个字面量
3. 如果一行的末尾它是break、continue、return等这些关键词的话，也会默认加分号
4. 如果一行的末尾它是以++、--、)、}、]，右侧的，它也会默认地加分号
func main()         //这里会自动加分号
{

}
